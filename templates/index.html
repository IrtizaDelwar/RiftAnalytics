Welcome to Rift Analytics!

Rift Analytics is a web application for the game League of Legends. On this web application we display various data and analytics for the game.

Features:

Ultimate Bravery

Are you ready for a challenge? Check out ultimate bravery! Click on the challenge button and follow the Champion, Summoner Spell, and Item Rules in your next match on Summoner's Rift.
